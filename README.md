## Getting started

> [!IMPORTANT]
> in `application.properties`
> ```
> #private token from ipinfo.io
> token.ip.info = ${TOKEN_IP_INFO}
> 
> #url frontend (http://localhost:5173)
> url.fe.cross.origin = ${URL_FE_CROSS_ORIGIN}
> ```
